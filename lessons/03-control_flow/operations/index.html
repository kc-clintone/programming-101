    ‚ûï Arithmetic in Bash¶

    üß† Learning Objectives¶

    By the end of this lesson, learners will:

    • Perform integer arithmetic using let, $(( )), and expr
    • Use increment (++) and decrement (--) operations
    • Understand Bash's math limitations (no float support)
    • Apply arithmetic in practical scripts

    üî¢ Bash and Math: Quick Overview¶

    Bash supports basic integer arithmetic, including:

    • Addition: +
    • Subtraction: -
    • Multiplication: *
    • Division: /
    • Modulo (remainder): %

    üìõ But: Bash does not support floating-point numbers ‚Äî math is limited to integers only!

    üßÆ 1. Using $(( )) ‚Äî Arithmetic Expansion (Preferred)¶

    This is the most common and modern way to do math in Bash.

    sum=$((5 + 3))
echo "Sum: $sum"

    You can use variables directly:

    a=10
b=4
result=$((a * b))
echo "Result: $result"

    ‚úÖ Works inline, easy to read, no need for quotes or escaping.

    üîß 2. Using let ‚Äî Arithmetic Operator (Also Common)¶

    The let command evaluates an arithmetic expression.

    let x=5+3
echo "x = $x"

    Or with variables:

    a=8
b=2
let result=a/b
echo "Result: $result"

    üîÅ Increment and Decrement with let¶

    count=0
let count++
let count+=2
let count--
echo "Count: $count"

    You can also use ++ and -- with $(( )), but not with expr.

    üßæ 3. Using expr ‚Äî Older Style (Still Works)¶

    The expr command does math but requires spaces between every token.

    result=$(expr 7 + 2)
echo "Result: $result"

    ‚ö†Ô∏è Important:¶

    • You must escape the * (multiplication) with \* because * is a shell wildcard:
    result=$(expr 6 \* 3)
echo "6 times 3 is $result"
    • No support for ++, --, or += style operators.

    üö´ Limitation: No Floats!¶

    Bash can‚Äôt handle decimals:

    a=5
b=2
echo $((a / b))    # Output: 2, not 2.5

    If you try:

    echo $((5 / 2))     # ‚Üí 2

    üí° Workaround:¶

    To use floating point math, use tools like:

    • bc (Basic Calculator):

    echo "scale=2; 5 / 2" | bc   # Output: 2.50
    * awk for inline math:

    echo | awk '{print 5/2}'     # Output: 2.5

    üìà Arithmetic Operators Summary¶

    Operator Description Example
    + Addition $((a + b))
    - Subtraction $((a - b))
    * Multiplication $((a * b))
    / Division (int only) $((a / b))
    % Modulo $((a % b))
    ++ / -- Increment / Decrement $((a++)), let a--
    += / -= Compound assignment let a+=2

    üîÇ Examples: Increment and Decrement¶

    counter=0
echo "Start: $counter"

((counter++))   # post-increment
echo "After ++: $counter"

((counter--))   # post-decrement
echo "After --: $counter"

((counter+=5))  # add 5
echo "After +=5: $counter"

    üß† Best Practices¶

    • Prefer $(( )) for arithmetic ‚Äî modern and clean
    • Use let for simple math or when incrementing directly
    • Avoid expr unless required for legacy scripts
    • Remember: Bash math is integer-only
    • Use bc if you need decimal support

    üß™ Mini Script Example¶

    #!/bin/bash

read -p "Enter a number: " num
echo "Square: $((num * num))"
